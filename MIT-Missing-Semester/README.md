# MIT-Missing-Semester

## 课程简介

这门课介绍了 CS 中基础到不能基础，也无比重要的一些工具。它们是  Shell、Git、Vim、SSH......

整个课程包括 11 个视频及其对应的阅读材料和习题。每个视频时长约 1 hour。
