# CS61A-Structure-and-Interpretation-of-Computer-Programs

## 课程简介

这门课是经典的伯克利 SICP python 版课程，MIT 与之对应的课程是 MIT 6.001（但这门课程已经不再开设了）。

CS61A 和 SICP 的中心思想都是抽象：即如何控制系统的复杂度。
